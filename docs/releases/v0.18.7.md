## Added
- Closed tabs can now be restored in the current process with `Ctrl+Shift+T` (`Cmd+Shift+T` on macOS), using LIFO order and fresh tab IDs.
- Added `docs/ARCHITECTURE_OVERVIEW.md` as a short map for the runtime flow, ownership boundaries, and change-type entrypoints.

## Changed
- Split the SDD, detailed design, and test planning docs into topic files while keeping the top-level documents as entrypoints.
- Expanded the Validation Matrix with a change-type checklist for docs-only, GUI, indexing, search, release/update, runtime config, and supportability work.
- Reordered TC / VM / SP tables and normative sections by ID, and added the ID Ordering check to the release preflight skill.

## Security
- query history は既定で平文永続化される。必要なら runtime config の `history_persist_disabled=true` を案内する。
- 自動更新の対象ビルドは埋め込み公開鍵で `SHA256SUMS.sig` を検証し、その後 `SHA256SUMS` の checksum を照合する。
- 当面の暫定運用として、macOS 配布物は未 notarized のまま publish する場合があり、その場合は本 release の `Known issues` または `Security` に明記する。

## Known issues
- notarization 環境が整うまで、macOS 配布物は未 notarized の場合がある。
