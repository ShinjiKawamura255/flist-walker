Distribution policy in this repository
--------------------------------------

To keep distributed builds compliant with the above notices:

- release archives include LICENSE.txt and THIRD_PARTY_NOTICES.txt
- macOS app bundles also include these notice files in Contents/Resources
- release directories include sidecar notice files for standalone binary assets

This notice file is a compliance aid for current packaged builds. If
dependencies change, this file and the release packaging rules must be updated
in the same change.
